# **App Name**: WorkspaceZ

## Core Features:

- Auth & Workspace Onboarding: Email-based registration with a streamlined choice to create a new workspace with a unique code or join an existing one as a member.
- Chat-to-Task Conversion: Create tasks directly from chat messages, automatically pre-filling the task title and description fields with the selected message content.
- Hybrid Lifecycle Management: Task tracking featuring dual-mode progress logic including a manual 0-100 slider or auto-calculation based on subtask completion status.
- Centralized Workspace Communications: Integrated real-time chat functionality that serves as the collaborative hub for team communication within each workspace.
- Unified Productivity Dashboard: Clean landing view containing summary cards for all task statuses, a visual activity feed, and nearest upcoming deadlines.
- Resource Vault: Shared workspace notes and 1GB per-workspace file storage with a visual usage meter for superadmins.
- Visual Scheduling: A monthly calendar interface displaying tasks by due date with distinct highlighting for overdue items.

## Style Guidelines:

- Primary color is a professional Vibrant Blue (#3B82F6). Background is an airy, light Blue-Grey (#F8FAFC) to minimize eye fatigue. Accent color is a bright Deep Cyan (#0EA5E9) for active states.
- Utilize 'Inter' sans-serif for its objective, machined look and exceptional legibility for high-density dashboard data.
- Outline icons with a consistent 2px stroke width, maintaining a minimalist 'less is more' approach to ensure a clutter-free sidebar.
- Modular grid layout featuring cards with 12px border radii for a soft, professional SaaS aesthetic and a slim, minimal sidebar.
- Smooth micro-interactions such as hover transitions for task items and subtle height expansion when viewing sub-tasks.
